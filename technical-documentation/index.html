Introduction

Control Structures

Control flow refers to the sequence a program will follow during its execution.

Conditions, loops, and calling functions significantly increase how a Python program is controlled.

There are three types of control structures in Python:

  • Conditional Statements - if, elif, else
  • Loops - for, while
  • Control Statements - break, continue, pass

Here are some examples:

Conditional Statements


          if x>0:
             print("positive")
          elif x==0:
             print("zero")
          else:
             print("negative")

Loops


          for i in range(n):
                print(i)
          while x < n:
                print(x)

Control Statements


          for i in range(5):
                if i==3:
                   break
                print(i)

          for i in range(5):
                if i==5:
                  continue
        
          for i in range(6):
                pass #acts as a place holder
Functions

A function is a block of code which only runs when it is called.

You can pass data, known as parameters, into a function.

A function can return a data as a result.

Creating a function

In Python a function is defined using the def keyword


            def my_function:
                 print("Hello!")

Calling a function

To call a function, use the function name followed by paranthesis


            my_function()
Exception Handling

Exception handling in Python allows you to manage and respond to unexpected events that occur during program execution.

Key concepts in exception handling include:

  • try: block where you attempt the operation that may raise an exception.
  • except: block to handle specific exceptions raised within the try block.
  • finally: block that executes regardless of whether an exception occurred or not, often used for cleanup actions.

Example:


              try:
                  numerator = 10
                  denominator = 0
                  result = numerator/denominator
                  print(result)
              except:
                  print("Error: Denominator cannot be 0.")
    
               finally:
                  print("This is finally block.")
